// Первая - сборка-утверждение, в которой собраны все этикетки из списка.

// Порядок действий скрипта следующий:
// открывается шаблон,
// помещается этикетка №1 и выравнивается по первому
// контуру высечки, помещается этикетка №2 и выравнивается по второму pathitem-у.
// И так до конца списка.
// Шаблоном для сборки-утверждения может служить укороченный вариант (имя файла-шаблона со словом short).
// Файлов сборок-утверждений может быть несколько, если в одной сборке не умещаются все этикетки.
// Синтаксис следующий: [номер паспорта]_UTV.pdf, [номер паспорта]_UTV_2.pdf и так далее.

// Вторая - сборка "Внимание! Новый вид!"

// Делается для того, чтобы при смотке оператор (смотчик) визуально видел, когда заканчивается один вид и начинается другой.
// Порядок создания сборки-внимания следующий:
// открывается short-шаблон,
// в него помещается файл ACHTUNG.eps, позиционируется
// по центру, уменьшается по ширине (при необходимости) и растягивается по всей длине сборки.
// Сохраняется с синтаксисом [номер паспорта]_ACHTUNG.pdf
